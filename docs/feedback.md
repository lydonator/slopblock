### üêõ Report a Bug

Found something broken? Help us fix it!

**GitHub Issues:** [Report a Bug](https://github.com/lydonator/slopblock/issues/new?template=bug_report.md)

**What to include:**
- Chrome version (e.g., Chrome 120)
- Extension version (e.g., 0.1.0)
- Steps to reproduce the bug
- Expected vs. actual behavior
- Screenshots or console errors (if applicable)

**Example:**
```
**Bug:** Warning icon doesn't appear on home feed

**Chrome Version:** 120.0.6099.129
**Extension Version:** 0.1.0

**Steps to Reproduce:**
1. Install SlopBlock
2. Go to YouTube home page
3. Report a video (becomes green)
4. Refresh page
5. Icon doesn't appear on thumbnail

**Expected:** Icon should appear on marked video thumbnail
**Actual:** No icon visible

**Console Errors:** None
```

## FAQ About Feedback

**Q: How do I find my extension version?**
A: Click the SlopBlock icon in Chrome ‚Üí Look at bottom of popup (e.g., "Version 0.1.0")

**Q: Can I submit anonymous feedback?**
A: Yes! GitHub Issues and Discussions allow anonymous posting (create a throwaway account if desired)

**Q: Will my feature request definitely be implemented?**
A: We can't promise to implement everything, but we read and consider all suggestions!

**Q: I reported a bug but got no response. Why?**
A: We try to respond to everyone, but some issues may take longer. Please be patient, and bump your issue after 7 days if no response.

**Q: Can I help maintain the project?**
A: Absolutely! Regular contributors may be invited to join the maintainer team. Reach out via GitHub!
